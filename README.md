# デバッガーズインターン

ゲーム会社のインターンとしてデバッグ業務を体験する短編ADVゲーム。

プレイヤーは夏休みの短期インターン生として、開発中ゲームの不具合調査・再現確認・報告を行いながら、デバッガーとしてのランクアップを目指す。

**プレイ時間：** 20〜40分  
**ジャンル：** 短編ADV / 推理・観察系
