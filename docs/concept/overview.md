# 概要・コアコンセプト

## 概要

『デバッガーズインターン』は、ゲーム会社のインターンとしてデバッグ業務を体験する短編ADVゲームである。

プレイヤーは夏休みの短期インターン生として、開発中ゲームの不具合調査・再現確認・報告を行いながら、デバッガーとしてのランクアップを目指す。

単なる「間違い探し」ではなく、

- 情報を整理する
- 状況を観察する
- 正しく伝える
- 曖昧さを潰す

といった、ソフトウェアテスト・QAの本質的な感覚を、ゲームとしてカジュアルに体験できる作品。

---

## コアコンセプト

### 「バグを見つける」ではなく「伝わる報告をする」

本作のテーマは、単に不具合を発見することではない。

- 何が起きたか
- どの条件で起きるか
- どのくらい危険か
- 再現するか

を整理し、開発側に**伝わる形で報告すること**に価値がある。

プレイヤーは、「観察して伝える」ことそのものをゲーム体験として味わう。
