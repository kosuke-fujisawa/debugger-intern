# バグの種類・難易度別分類

## 初級

| 種類 | 説明 |
|------|------|
| 誤字 | テキストの文字ミス |
| UI崩れ | レイアウト・表示位置の乱れ |
| 音ズレ | SEやBGMのタイミングずれ |
| 軽微な表示不具合 | 色・サイズなどの視覚的ミス |

発見しやすく、再現も容易。報告の練習に適している。

---

## 中級

| 種類 | 説明 |
|------|------|
| 特定条件で進行不能 | 特定操作をすると先に進めなくなる |
| セーブ破損 | セーブデータが壊れる・読み込めない |
| パラメータ異常 | 数値・ステータスが意図しない値になる |

再現条件の切り分けが必要になり始める難易度帯。

---

## 上級

| 種類 | 説明 |
|------|------|
| 再現条件が複雑 | 複数の操作・状態が組み合わさって発生 |
| 発生頻度が低い | ランダム性があり毎回は起きない |
| 他要素と絡む | 別のバグや機能との相互作用で起きる |

「どこまで絞り込めるか」の推理力が試される。
報告の精度がスコアに大きく影響する。
